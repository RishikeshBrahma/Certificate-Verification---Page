Welcome to Certificate Verification System

This system allows you to upload certificate data in CSV format, generate QR codes for verification, and download associated CSV files by scanning QR codes or using manual links.

How to Upload and Create the Correct CSV File

To use this system effectively, follow these steps:

  1. Create a CSV File: Use Excel or Notepad to create a CSV file.
  2. Required Columns:
    • certificate_id
    • recipient_name
    • course_title
    • issue_date
  3. Save as: .csv extension (e.g., certificates.csv)
  4. Upload: Go to the Upload Page and upload your file.
